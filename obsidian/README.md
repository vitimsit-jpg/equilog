# Equilog — Vault Obsidian

Base de connaissances du projet Equilog, versionnée avec Git.

## Structure

| Dossier | Contenu |
|---------|---------|
| `architecture/` | Stack technique, structure app, composants clés |
| `features/` | Documentation des fonctionnalités |
| `decisions/` | ADR (Architecture Decision Records) |
| `roadmap/` | Planification et suivi des versions |
| `bugs/` | Tickets de bugs et investigations |
| `daily/` | Notes quotidiennes de développement |

## Utilisation

1. Ouvrir ce dossier comme vault dans Obsidian
2. Les liens `[[wikilinks]]` connectent les notes entre elles
3. Le vault est versionné avec Git (push/pull via le plugin Obsidian Git ou manuellement)
